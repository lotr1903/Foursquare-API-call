## Foursquare API call example 

## Running project
First use command `git clone` or download button to get repository to your local machine.

Then to run the project in your local enviroment, first execute `npm install` in the project directory to get neccessary npm packages.

When all the packages installed successfully, you can execute command `npm start`.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Testing 
Execute `npm test` to see test results of the project. <br>

### Building 
Execute `npm build` to build project to deploy it. <br>
